Hexoの記事をGitHub Pagesにデプロイする

GitHub側での作業

公開用の.github.ioリポジトリを作成する。

Hexo側での設定

Hexo Ver3.0.1で確認しています。

プラグインのインストール

Ver3.0.0から、デプロイには別途npmモジュールのインストールが必要になっていて、hexo-deployer-gitをインストール。

備考：--saveオプションをつけるとpackage.jsonのdependenciesに、インストールしたnpmモジュールを自動的に記入してくれます。

_config.yml

_config.ymlのdeploy部分を編集。

branchとmessageは任意で入力。branchはGitHubを使用していれば自動で検出されます。

messageは何も入れなければ、デフォルトでSite updated: {{ now("YYYY-MM-DD HH:mm:ss") }}が入る。

ふと…

なんだか結局、Hexoのドキュメントをコピーした感じになったので、公式のドキュメントを見るのがよさそう。
